# Comparing Classifiers: Bank Marketing Campaign

## Business Problem
Predict whether a customer will subscribe to a term deposit.

## Models Compared
1. Logistic Regression
2. K-Nearest Neighbors
3. Decision Tree
4. Support Vector Machine

## Results
Logistic Regression achieved the best balance between performance and interpretability.

## Key Findings
- Previous campaign success strongly predicts subscription.
- Longer customer interactions increase conversion likelihood.
- Targeted outreach can reduce marketing costs.
